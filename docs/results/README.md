# Final Coursework Results

This folder contains the curated output from the best observed CIFAR-10 run:

```text
python cifar10_cnn.py --architecture research_cnn --epochs 45 --strong-augmentation --robustness-samples 1000 --output-dir artifacts_cnn_45
```

The model checkpoint and raw console logs are intentionally excluded because
they are generated training files. The source code can regenerate them.

## Summary

| Metric | Value |
| --- | ---: |
| Best validation accuracy | `0.8256` |
| Best validation epoch | `45` |
| Final test accuracy | `0.8134` |
| Final test loss | `0.6764` |

Compared with the earlier 30-epoch baseline, the 45-epoch run improved test
accuracy from `0.7936` to `0.8134`.

## Files

- `research_report.md`: report-ready summary generated by the training script.
- `metrics.json`: final train, validation, and test metrics.
- `per_class_accuracy.json`: accuracy for each CIFAR-10 class.
- `confusion_matrix.json`: raw confusion matrix data.
- `robustness.json`: noise, brightness, and occlusion robustness results.
- `training_log.csv`: epoch-by-epoch training history.
- `config.json`: exact reproducible run configuration.
- `plots/training_curves.png`: train/validation accuracy and loss curves.
- `plots/confusion_matrix.png`: class-level confusion matrix.
- `plots/prediction_examples.png`: example predictions with confidence.
- `plots/robustness.png`: accuracy under image corruptions.
- `plots/gradcam_examples.png`: Grad-CAM interpretability examples.
